# Effort

The effort level for reasoning (none/minimal=fast, low/medium=balanced, high/xhigh=thorough)

## Example Usage

```typescript
import { Effort } from "@r9s/sdk/models";

let value: Effort = "minimal";
```

## Values

```typescript
"none" | "minimal" | "low" | "medium" | "high" | "xhigh"
```
